## How To Use This Package
1. Read the master roadmap first.
2. Execute plans in order: 02 -> 03 -> 04.
3. Use [99-coverage-check.md](99-coverage-check.md) before execution kickoff and again at the end of implementation.
4. If capacity drops, cut from later phases first; do not cut instrumentation, trust, or quest-posting conversion foundations.

## Planning Principles
- Prioritize marketplace liquidity over ornamental feature growth.
- Ship instrumentation before optimization.
- Improve buyer trust before expanding top-of-funnel spend.
- Prefer reusable surfaces over one-off promo pages.
- Keep each weekly milestone independently demoable.
